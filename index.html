AZ Central News Scraper

The AZ Central News Scraper is an application which scrapes the AZ Central website and displays articles to the user. This app is a work in progress, the user authentication is currently in development. Having user authentication will allow the user to save articles and comment on them as well.

    Technology used:
  • React.js
  • Express.js
  • Node.js
  • MongoDB
  • Cheerio (npm)

Class Group Generator

The Skill-based group generator. This application was originally developed to assist Trilogy Education Instructional Staff in equally skilled classroom group generation. The initial app was developed as a node application that takes in an array of students and their skill and returns equal groups. A user interface is currently being developed in React and will have the ability for students to enter their own name and skill level.

    Technology used:
  • React.js
  • Express.js
  • Node.js
  • MongoDB
  • SCSS

Skelly

Skelly is an online tool that allows you to customize a skeleton for various project types. As a proof of concept, it uses various NPM packages to allow a user to customize the start of a project through a web interface. The structure is then delivered to the user as a zip file.

    Technologies used:
  • Handlebars.js
  • Express.js
  • Node.js
  • MySQL
  • Materialize.css
